LABEL STUDIO WITH VIRTUAL PAPER AND ENVELOPE PRINTER

This is a separate merged PWA. The supplied original app and the tested standalone virtual-paper app were not modified.

VIRTUAL PAPER LOGIC
- Select A4 or A5 as the paper reported to the printer.
- Enter the real label width and height.
- Place the label strip on Auto, Left, Right, Top or Bottom.
- Auto uses Left for portrait and Top for landscape.
- Left/Right strips begin after a 20 mm top feed gap.
- Top/Bottom strips begin after a 20 mm left feed gap and rotate the complete label by 90 degrees.
- Print exported files at 100% / Actual size. Disable Fit to page.

SOURCES AND OUTPUTS
- Print current TO/FROM app data, including formatting, QR codes and images.
- Or import PNG, JPG, WebP, GIF or PDF labels.
- Export virtual A4/A5 output as PDF or PNG.

ENVELOPE PRINTER
- Enter any envelope or label width and height in millimetres.
- Set independent top, right, bottom and left margins; dragged items remain inside the safe area.
- TO starts in the centre; FROM starts at the top-left.
- Drag TO, FROM, both QR codes and both images independently with mouse or touch.
- Hide or show each item, change text/QR/image sizes and reset positions.
- Export the exact entered size as PDF or PNG. Blue editing guides are not exported.

ONE PREVIEW WINDOW
- The same Preview window is used for normal labels, Virtual Paper and Envelope Printer.
- Inspect mode shows live cursor coordinates from every edge of the active printable area.
- QR and image windows show their dimensions and position, and can be dragged with mouse or touch.
- Envelope TO/FROM blocks can also be dragged and display their window dimensions.
- Final mode shows the clean print result without rulers, coordinates or item guides.

FILENAMES AND HISTORY
- Standard labels retain the existing name, paper size and orientation convention.
- Virtual Paper files use Name_Virtual_A4-or-A5_LabelDimensions.format.
- Envelope files use Name_Envelope_Dimensions.format.
- Saving PDF or PNG adds the complete workspace to History, including dimensions, margins, visibility, QR/image positions, envelope positions and Virtual Paper settings.
- Restoring a History entry returns to the correct Label, Virtual Paper or Envelope workspace.

PRINTED CONTACT MARKERS
- Mobile prints as icon + (M), telephone as icon + (T), WhatsApp as icon + (Whatsapp), and email as icon + (E).
- QR payload and copy-to-notes behaviour are unchanged.

INSTALLATION
Host this folder on HTTPS or localhost, open index.html and install it from the browser. PDF import is bundled for offline use.
